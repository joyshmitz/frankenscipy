# fsci-constants COVERAGE

Reality-check audit of `fsci_constants::value()` against
`scipy.constants.physical_constants` (CODATA 2022, scipy 1.13).

Generated by frankenscipy-wada. Refresh with the audit script:
`python3 crates/fsci-constants/audit/audit_constants.py`.

## Summary

| Status | Count | % of scipy table |
|--------|------:|----------------:|
| ✅ covered  |  55 | 12.4% |
| ❌ missing  | 390 | 87.6% |
| **scipy.constants.physical_constants entries** | **445** | **100%** |

The 12.4% covered set is intentionally curated for the highest-traffic
physics constants (CODATA-exact SI values, fundamental particle masses,
g-factors, radiation constants). The long tail (nuclear masses in u,
isotope-specific magnetic moments, parent / child mass ratios) is
deferred to per-domain follow-up beads — file as needed.

fsci-constants additionally exposes ~70 module-level constants that
are NOT in `physical_constants` but ARE in the top-level
`scipy.constants` namespace (math constants, SI prefixes, conversion
factors like `pound`, `atm`, `light_year`). Those are tracked
separately in P2C-016 as direct-name fixture cases — not in this
table.

## Missing — top categories

The 390 missing entries break down loosely as (from the audit script's
default top-50 dump and a manual scan):

| Category | rough count | typical pattern |
|----------|------------:|-----------------|
| `... in eV` / `... in u` energy-unit forms | ~55 | mass × c² conversions of the masses we already have |
| atomic / nuclear isotope masses & magnetic moments | ~70 | low daily traffic; file P3 if a consumer asks |
| `Compton wavelength of <particle>` and `<particle>-<particle>2 mass ratio` | ~60 | derived from existing masses; can be computed at call site |
| 2018-vs-2022 dual-CODATA entries (deprecated tags) | ~20 | scipy keeps both during CODATA transitions |
| `vacuum mag/elec permeability` synonyms | ~15 | aliases for already-covered `mu_0` / `epsilon_0` |
| `conventional value of ...` | ~10 | pre-2019 SI redefinition values |
| niche thermal / electrochemical constants | ~150 | application-specific; expand on demand |

## Top-priority follow-up beads

The first long-tail ratchet is now covered: in-MeV particle energies,
electron/proton/neutron/muon/tau Compton wavelengths, and common
particle mass ratios are wired through `value()` and P2C-016.

Remaining long-tail categories (atomic radii, isotope masses,
conventional values, magnetic moments, in-eV/in-u aliases) — file as
they get requested. The audit script makes future re-runs quick.
